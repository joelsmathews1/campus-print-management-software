# Get Your Print

A simple campus printing web application built using React and Supabase.

This project allows students to upload print jobs and staff to manage and update their status.

---

## Features

- Login and Signup (with role selection: Student / Staff)
- Student Dashboard
  - Submit print job
  - Upload file
  - Track status
- Staff Dashboard
  - View all print jobs
  - Update job status (Pending / Completed)

---

## Tech Stack

- Frontend: React (Vite)
- Backend: Supabase (PostgreSQL)
- Storage: Supabase Storage
- Version Control: Git + GitHub
